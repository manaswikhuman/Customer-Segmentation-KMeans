# Customer-Segmentation-KMeans

This is a beginner-friendly project using **Python** and **K-Means Clustering** to group customers based on their income and spending habits.

### Features
- Uses `Scikit-learn` for the clustering model.
- Uses `Matplotlib` for visual results.
- Simple and easy-to-understand code.

### How to Run
1. Install requirements: `pip install pandas matplotlib scikit-learn`
2. Run the script: `python main.py`
